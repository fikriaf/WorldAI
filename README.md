From Absolute Emptiness to a Self-Aware Civilization. A simulation where complex systems emerge from simple rules — no shortcuts, no hardcoding, pure emergence.

## Vision

> *"In the beginning, there was nothing. Then there were rules. From rules, complexity emerged. From complexity, life arose. From life, thought appeared. From thought, a question was born: Who am I?"*

World.AI is a self-evolving civilization simulation built on emergent behavior, where intelligence arises from the interaction of simple agents rather than pre-programmed logic.

## Architecture

- Agent-based simulation engine
- Emergent social structures and governance
- Self-aware civilization dynamics
- Alembic database migrations

## Getting Started

```bash
# Install dependencies
pip install -r requirements.txt

# Run database migrations
alembic upgrade head

# Start simulation
python -m packages.main
```
